Theorem 17: Hierarchical Orthogonality Theorem

Statement

Suppose there exist the vector representations for all the following binary features. Then, we have that
  1. $\bar\ell_{w_1} - \bar\ell_{w_0}$ is a linear representation $\bar{\ell}_{\ConceptDirMath{w_0}{w_1}}$ defined in Definition 4;
  2. $\bar\ell_{w} \perp \bar\ell_{z} - \bar\ell_w$ for $z \prec w$;
  3. $\bar{\ell}_{w} \perp \bar\ell_{z_1} - \bar\ell_{z_0}$ for $Z \in_R \{z_0, z_1\}$ subordinate to $W \in_R \{\ConceptValue{not_w}, \ConceptValue{is_w}\}$;
  4. $\bar{\ell}_{w_1} - \bar{\ell}_{w_0} \perp \bar\ell_{z_1} - \bar\ell_{z_0}$ for $Z \in_R \{z_0, z_1\}$ subordinate to $W \in_R \{w_0, w_1\}$; and
  5. $\bar\ell_{w_1}-\bar\ell_{w_0} \perp \bar\ell_{w_2} - \bar\ell_{w_1}$ for $w_2 \prec w_1 \prec w_0$.
