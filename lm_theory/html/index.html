Introduction

Welcome to the repository of theoretical results on transformers and language models. This site aims to organize and present definitions, axioms, lemmas, theorems, and corollaries related to these topics.

Data Summary

This repository includes results extracted from several key papers. Each result is accompanied by references to the original publications and enriched with explanations and proofs.

How the Website is Built

This website is built using a workflow at Lunar for automatically extracting and processing theoretical results from academic papers. The extraction process involves parsing LaTeX from PDFs and generating enriched content using GPT-4o and formal verification systems like Isabelle/HOL.
